# 👑 Команды администратора AutoScaleX Pro 2.2

## Основные команды

### `/start`
Запуск бота и главное меню
- Администратор видит приветствие и может использовать `/admin`

### `/admin`
**Админская панель** - управление рефералами
- Открывает меню управления рефералами
- Доступна только администратору (TG_ADMIN_ID)

### `/set_api`
Настройка API ключей BingX
- Введите API Key
- Введите Secret
- Ключи сохраняются для текущего пользователя

---

## Админское меню (`/admin`)

### 📋 Список рефералов
Показывает всех зарегистрированных рефералов:
- UID реферала
- User ID в Telegram
- Username

### ➕ Добавить реферала
Добавление нового реферала в систему:
1. Нажмите "➕ Добавить реферала"
2. Введите UID реферала
3. Реферал будет добавлен в систему

**Пример:**
```
UID: 12345678
```

### ➖ Удалить реферала
Удаление реферала из системы:
1. Нажмите "➖ Удалить реферала"
2. Введите UID реферала
3. Реферал будет удален

---

## Главное меню (доступно всем, включая админа)

### ▶️ Старт
Запуск торгового бота
- Создает новую сетку ордеров
- Начинает торговлю

### ⏸️ Пауза
Приостановка торговли
- Ордера остаются активными
- Новые действия не выполняются

### 🛑 Стоп
Остановка торговли
- Отменяет все открытые ордера
- Полная остановка бота

### 💰 Баланс
Показывает:
- Баланс USDT (или другой quote валюты)
- Баланс базовой монеты (ETH, BTC и т.д.)
- Общий equity
- Прибыль и ROI
- Profit Bank
- Текущая цена
- Торговая пара

### 📋 Ордера
Показывает:
- Количество открытых BUY ордеров
- Количество открытых SELL ордеров
- Количество исполненных BUY
- Количество исполненных SELL

---

## Меню настроек

### 📌 Выбрать монету
Выбор торговой пары:
- Популярные пары: BTC-USDT, ETH-USDT, BNB-USDT, SOL-USDT и др.
- Возможность ввести пару вручную
- При смене пары все ордера по старой паре отменяются

### 📏 Шаг сетки
- **0.75%** - стандартный шаг
- **1.5%** - увеличенный шаг

### 💵 Размер ордера
- **10 USDT**
- **20 USDT**
- **50 USDT**

### 🔨 Построить сетку
Ручное построение сетки ордеров:
- Отменяет все существующие ордера
- Создает новую сетку согласно текущим настройкам

### 🔄 Ребалансировать SELL
Перестройка SELL сетки от VWAP:
- Создает 3 SELL ордера от средней цены покупки
- Уровни: VWAP × (1 + 1.5%), VWAP × (1 + 2.5%), VWAP × (1 + 3.5%)
- Весь объем распределяется равномерно

## Примеры использования

### Добавление нового реферала:

1. Отправьте `/admin`
2. Нажмите "➕ Добавить реферала"
3. Введите UID (например: `12345678`)
4. Реферал добавлен ✅

### Просмотр списка рефералов:

1. Отправьте `/admin`
2. Нажмите "📋 Список рефералов"
3. Увидите всех рефералов с их данными

### Удаление реферала:

1. Отправьте `/admin`
2. Нажмите "➖ Удалить реферала"
3. Введите UID реферала
4. Реферал удален ✅

### Настройка торговли:

1. Отправьте `/start`
2. Нажмите "⚙️ Настройки"
3. Выберите нужные параметры:
   - Шаг сетки
   - Размер ордера
   - Торговую пару
4. Нажмите "🔨 Построить сетку" для применения

---

## Важные замечания

1. **Администратор имеет доступ ко всем функциям** - как к админским, так и к обычным
2. **Рефералы проверяются по UID** - каждый реферал должен иметь уникальный UID
3. **API ключи настраиваются индивидуально** - каждый пользователь (включая админа) настраивает свои ключи через `/set_api`
4. **Состояние сохраняется автоматически** - все настройки и ордера сохраняются в файлы

---

## Файлы данных

- `referrals.json` - список рефералов
- `user_states/user_{user_id}.json` - состояние каждого пользователя
- `logs/bot_YYYYMMDD.log` - логи работы бота

---

## Безопасность

- Только пользователь с `TG_ADMIN_ID` имеет доступ к админским функциям
- API ключи хранятся в зашифрованном виде в файлах состояния
- Каждый пользователь работает со своими API ключами
