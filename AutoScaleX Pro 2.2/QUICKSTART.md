# Быстрый старт AutoScaleX Pro 2.2

## 1. Установка

```bash
# Установите зависимости
pip install -r requirements.txt

# Создайте .env файл
cp env.example .env
```

## 2. Настройка .env

Откройте `.env` и заполните:

```env
TG_TOKEN=ваш_telegram_бот_токен
TG_ADMIN_ID=ваш_telegram_id
```

Для получения Telegram ID используйте бота @userinfobot

## 3. Запуск

```bash
python main.py
```

## 4. Первый запуск (для администратора)

1. Откройте Telegram бота
2. Отправьте `/start`
3. Отправьте `/admin`
4. Добавьте рефералов через "➕ Добавить реферала"
5. Введите UID реферала

## 5. Для пользователей

1. Отправьте `/start`
2. Введите ваш UID (должен быть добавлен администратором)
3. Отправьте `/set_api` и введите:
   - BingX API Key
   - BingX Secret
4. Используйте меню для управления ботом

## Команды

- `/start` - Начать работу с ботом
- `/admin` - Админ панель (только для администратора)
- `/set_api` - Настроить API ключи

## Меню бота

- **▶️ Старт** - Запустить торговлю
- **⏸️ Пауза** - Приостановить торговлю
- **🛑 Стоп** - Остановить и отменить все ордера
- **💰 Баланс** - Показать баланс и прибыль
- **📋 Ордера** - Показать количество открытых ордеров
- **⚙️ Настройки** - Настройка параметров:
  - Шаг сетки (0.75% или 1.5%)
  - Размер ордера (10, 20, 50 USDT)
  - Построить сетку
  - Ребалансировать SELL

## Важно

- Все ордера сохраняются автоматически
- При перезапуске состояние восстанавливается
- Бот автоматически перезапускается при падении (до 10 попыток)
