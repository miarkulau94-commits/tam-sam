# 🤖 Настройка команд в BotFather

## Инструкция по настройке меню команд

### Шаг 1: Откройте BotFather

1. Найдите бота [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/start`

### Шаг 2: Выберите вашего бота

1. Отправьте `/mybots`
2. Выберите вашего бота из списка
3. Нажмите "Bot Settings" → "Menu Button" → "Edit Bot Commands"

**Или напрямую:**

1. Отправьте `/setcommands`
2. Выберите вашего бота из списка

### Шаг 3: Вставьте команды

Скопируйте и вставьте следующий текст:

```
start - Запустить бота и открыть главное меню
admin - Админская панель (только для администратора)
set_api - Настроить API ключи BingX
```

### Шаг 4: Подтвердите

BotFather подтвердит, что команды установлены.

---

## Альтернативный способ (через файл)

1. Откройте файл `BOTFATHER_COMMANDS.txt`
2. Скопируйте содержимое
3. Вставьте в BotFather при настройке команд

---

## Описание команд

### `/start`
Запускает бота и открывает главное меню с кнопками:
- Старт/Пауза/Стоп
- Баланс
- Ордера
- Настройки

**Использование:** `/start`

---

### `/admin`
Открывает админскую панель для управления рефералами.
Доступна только пользователю с ID, указанным в `TG_ADMIN_ID`.

**Функции:**
- Просмотр списка рефералов
- Добавление нового реферала
- Удаление реферала

**Использование:** `/admin`

---

### `/set_api`
Настройка API ключей BingX для текущего пользователя.

**Процесс:**
1. Отправьте `/set_api`
2. Введите ваш BingX API Key
3. Введите ваш BingX Secret
4. Ключи сохраняются автоматически

**Использование:** `/set_api`

---

## Важно

- Команды работают только для зарегистрированных пользователей
- Рефералы должны быть добавлены администратором через `/admin`
- API ключи настраиваются индивидуально для каждого пользователя

---

## Проверка

После настройки команд:
1. Откройте вашего бота в Telegram
2. Нажмите на кнопку меню (три линии) рядом с полем ввода
3. Вы должны увидеть список команд

---

## Дополнительные команды (опционально)

Если хотите добавить больше команд, можете использовать:

```
start - Запустить бота и открыть главное меню
admin - Админская панель (только для администратора)
set_api - Настроить API ключи BingX
help - Показать справку
status - Показать статус бота
balance - Показать баланс
orders - Показать открытые ордера
```

**Примечание:** Дополнительные команды (`help`, `status`, `balance`, `orders`) нужно будет добавить в код бота, если их еще нет.
