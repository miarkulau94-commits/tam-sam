AutoScaleX — алгоритмический торговый бот для спотовой торговли на BingX

Архитектура торговой системы:
• Grid Trading с динамической ребалансировкой ордеров
• Pyramiding механизм для масштабирования позиций через реинвестирование прибыли
• DCA стратегия для усреднения входа при волатильности
• FIFO-учет позиций для корректного расчета налогооблагаемой прибыли

Технические характеристики:
• Асинхронная архитектура на Python для высокопроизводительной обработки ордеров
• Поддержка всех спотовых торговых пар биржи BingX
• Настраиваемые параметры сетки: шаг 0.75%–1.5%, размер ордера 10–100 USDT
• Масштабируемая сетка до 130 BUY и 5 SELL ордеров с автоматическим размещением

Управление и надежность:
• Web-интерфейс через Telegram Bot API с полным контролем операций
• Персистентное хранение состояния с автоматическим восстановлением после перезапуска
• Автоматический рестарт с экспоненциальной задержкой (до 10 попыток)
• Интеллектуальная ребалансировка SELL-сетки от VWAP с учетом текущих позиций

Аналитика и отчетность:
• Мониторинг баланса и P&L в реальном времени
• Экспорт истории сделок в CSV формат для внешнего анализа
• Встроенная статистика с метриками производительности стратегии

Безопасность:
• Реферальная система для контроля доступа
• Шифрование хранения API credentials
• Circuit breaker паттерн для защиты от каскадных сбоев API
• Rate limiting с экспоненциальным backoff

Инициализация: /start
