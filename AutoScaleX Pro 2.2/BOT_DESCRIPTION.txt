спотовой торговли на бирже BingX с интеллектуальным управлением 
позициями и максимизацией прибыли.

📊 Многостратегический подход
   Комбинирует три проверенные торговые стратегии:
   • Grid Trading — автоматическая сетка ордеров для 
     захвата волатильности рынка
   • Pyramiding — умное масштабирование позиций через 
     реинвестирование полученной прибыли
   • DCA — стратегия усреднения для снижения рисков 
     при волатильных движениях

⚡ Высокая производительность
   Асинхронная архитектура обеспечивает:
   • Мгновенное размещение и отмена ордеров
   • Обработку множества операций одновременно
   • Минимальные задержки при работе с API биржи

🔄 Надежность и отказоустойчивость
   • Автоматическое сохранение состояния каждую секунду
   • Восстановление всех позиций после перезапуска
   • Автоматический рестарт при сбоях (до 10 попыток)
   • Защита от перегрузок через circuit breaker

🎯 Автоматизация на каждом этапе:
   
   При запуске:
   → Автоматическая рыночная покупка для SELL-сетки
   → Создание 5 SELL ордеров с прибылью от текущей цены
   → Построение BUY сетки с учетом баланса
   
   При исполнении BUY:
   → Немедленное создание SELL ордера с заданной прибылью
   → Добавление позиции в FIFO-очередь для точного учета
   
   При исполнении SELL:
   → Расчет прибыли по методу FIFO (First In, First Out)
   → Автоматический Pyramiding — создание нового BUY 
     при накоплении достаточной прибыли
   → Ребалансировка при исполнении всех SELL ордеров
   
   Критические ситуации:
   → Уведомление при истощении баланса
   → Создание критической SELL-сетки от VWAP
   → Предупреждения о необходимости пополнения
