ОСОБЕННОСТИ AutoScaleX:

🔹 Три стратегии в одном:
  • Grid Trading — автоматическая сетка ордеров
  • Pyramiding — расширение позиций через реинвестирование прибыли
  • DCA — усреднение цены входа при падении

🔹 Технические преимущества:
  • Асинхронная архитектура — высокая производительность
  • Автосохранение состояния — восстановление после сбоев
  • Автоперезапуск — устойчивость к падениям
  • FIFO-учет — корректный расчет прибыли по позициям

🔹 Гибкая настройка:
  • Любая торговая пара BingX (BTC, ETH, SOL и др.)
  • Настраиваемый шаг сетки: 0.75% или 1.5%
  • Размер ордера: 10/20/50/75/100 USDT
  • Масштабируемая сетка: до 130 BUY ордеров

🔹 Умная логика:
  • Автоматическая покупка для SELL-сетки (5 ордеров + резерв)
  • Ребалансировка SELL от VWAP при исполнении всех ордеров
  • Критический уровень с предупреждением при истощении баланса
  • Pyramiding: новый BUY создается автоматически при накоплении прибыли

🔹 Удобство управления:
  • Полный контроль через Telegram
  • Мониторинг баланса и прибыли в реальном времени
  • История сделок в CSV
  • Детальная статистика торговли

🔹 Безопасность:
  • Реферальная система доступа
  • Шифрованное хранение API ключей
  • Circuit breaker — защита от перегрузок API
  • Rate limiting с экспоненциальной задержкой
